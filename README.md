## Install Dependencies
```shell
npm install
```

## Development

#### Frontend

```shell
npm run dev -- --host
```

#### Backend

```shell
npm run serve:dev:host
# or
npm run serve:dev
```

## Production

#### builds frontend and compiles backend typescipt

```shell
npm run build
```

#### Frontend

```shell
npm run preview -- --host
```

#### Backend

```shell
npm run serve -- --host
```
